GitNotes

Without a token, GitNotes works on public repositories at GitHub's unauthenticated limit of 60 requests/hour. A fine-grained personal access token with the repository Contents (read) permission raises that to 5,000 requests/hour and enables private repositories. The token is stored only in this extension's storage and is sent only to the GitHub API.

Note: git notes are not pushed by default. For the extension to see them, push them to GitHub with git push origin 'refs/notes/*'.
